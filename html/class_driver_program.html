Detailed Description

This is the detailed description for the DriverProgram class, this is the main class used to instantiate from the Listener classes; MainListener and HTMLListener and uses some driver functions to perform the required tasks of generating injected java code, running the generated intermediate java code, and generating html output to color entered and not entered blocks.

The main function is used as a driver function for the whole program, it reads the input java file, generates the lexer, generates tokens, builds parser for these tokens, and generates a parse tree, then it calls generateIntermediateCode to generate java code, runIntermediateCode to run the generated code, and generateHtmlOutput to generate output html file indicating which blocks were entered and which were not.
